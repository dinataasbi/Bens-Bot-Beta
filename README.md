# Bens Bot Beta - Repository

## Cara menginstal script dan menjalankanya
```bash
$ git clone https://github.com/rezadinata18/Bens-Bot-Beta.git
$ cd Bens-Bot-Beta
$ yarn
$ npm start 
```
